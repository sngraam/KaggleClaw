Use this tool to search the web for information about the competition, ML techniques, and relevant research.

## Usage

Send a plain text search query. You will receive up to 5 results with titles, snippets, and URLs.

## When to search

- Before starting: research the competition topic, similar past competitions
- Look for top public Kaggle notebooks and discussions for this competition
- Research domain knowledge (e.g., fraud detection patterns, medical imaging tips)
- Find ML papers, benchmarks, or technique comparisons
- Look up unfamiliar library APIs or error messages

## Good query examples

```
Kaggle titanic feature engineering top solutions
XGBoost vs LightGBM binary classification performance
handling imbalanced datasets SMOTE alternatives
time series cross-validation leakage prevention
```

## Tips

- Be specific: include the competition name or domain for better results
- After browsing, synthesize findings into your plan before running code
- Verify important facts (metric names, dataset columns) with the actual files
